Dashboards (Analytics) let you create custom visualizations of your monitoring data, organize them into collections, and share them with your team. You can build dashboards from scratch, favorite frequently-used ones, and control who has access.

## Creating dashboards

Click the "Create dashboard" button to open the creation dialog. Name your dashboard, select the products (sources) to include, and add a description. Dashboards use global filters based on the products you select, so you can focus on specific data.

Once created, the dashboard opens in edit mode. Build your visualization by adding charts, tables, and metrics from your monitoring data.

## Dashboard organization

Organize dashboards into nested collections (folders) to keep related dashboards together. Use the breadcrumb navigation to move through collections. Create a new collection at any level to group dashboards by team, project, or purpose.

Drag and drop dashboards between collections to reorganize them. Move a dashboard to a different collection using the move modal in the dashboard actions menu.

## Finding dashboards

Search across all dashboards by name and metadata using the full-text search bar. Search results appear instantly as you type.

Filter dashboards by **source** (products used in their global filters) or by **owner** (who created the dashboard) using the multi-select dropdown filters. Combine filters to narrow down results further.

## Favorite dashboards

Mark frequently-used dashboards as favorites by clicking the star icon. Favorites appear at the top of the list and are saved in your user preferences, making them easy to find.

## Sharing and access control

Share a dashboard with team members or publicly. Click the share action in the dashboard menu to open the sharing dialog. Generate a shareable link and optionally protect it with a password. Deactivate sharing at any time to revoke access.

View the owner of each dashboard (the creator) in the table. The "Last modified" column shows when the dashboard was last updated with a relative timestamp (e.g., "2 hours ago").

## Locking dashboards

Lock a dashboard to prevent accidental edits. Locked dashboards show a lock icon in the table. Unlock a dashboard from the actions menu when you need to make changes.

## Deleting and trash

Delete a dashboard using the actions menu. Deleted dashboards move to the trash collection and can be restored or permanently deleted. Access the trash by navigating to the trash collection if you need to recover a dashboard.
