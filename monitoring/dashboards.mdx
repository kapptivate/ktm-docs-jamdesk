Build custom dashboards to visualize your monitoring data exactly how you need it. Organize dashboards into collections, share them with your team, mark favorites, and lock dashboards to prevent accidental changes.

## Building custom dashboards

Click the "Create dashboard" button to open the creation dialog. Name your dashboard, select the products (sources) to include, and add a description. Dashboards use global filters based on the products you select, so you can focus on specific data.

Once created, the dashboard opens in edit mode. Add widgets by selecting metrics from your monitoring data, configuring charts, and customizing time ranges and filters.

### Dashboard widgets and metrics

Each dashboard widget displays a specific metric or visualization. Configure widgets with:
- **Metrics**: select from available KPIs, performance data, and monitoring metrics
- **Time ranges**: set the default date range for the widget
- **Filters**: apply additional scope filters by test or product

### Dashboard variables and filters

Create custom variables (text fields, dropdowns, multi-select lists) to let dashboard viewers filter data dynamically. Variables can reference global scope (tests, products) and support default values. All widgets using a variable update when the viewer changes its value.

Global filters apply to all widgets on the dashboard and are set during dashboard creation.

## Auto-generated dashboards

Tests with monitoring enabled automatically generate dashboards showing metrics and performance data. These auto-generated dashboards help you quickly visualize test results without manual setup. Find auto-generated dashboards in the **Auto Dashboards** section, searchable and filterable by test and product.

## Organizing dashboards

Organize dashboards into nested collections (folders) to keep related dashboards together. Use the breadcrumb navigation to move through collections. Create a new collection at any level to group dashboards by team, project, or purpose.

Drag and drop dashboards between collections to reorganize them. Move a dashboard to a different collection using the move modal in the dashboard actions menu.

## Finding dashboards

Search across all dashboards by name and metadata using the full-text search bar. Search results appear instantly as you type.

Filter dashboards by **source** (products used in their global filters) or by **owner** (who created the dashboard) using the multi-select dropdown filters. Combine filters to narrow down results further.

## Favorite dashboards

Mark frequently-used dashboards as favorites by clicking the star icon. Favorites appear at the top of the list and are saved in your user preferences, making them easy to find.

## Sharing dashboards

Share a dashboard with team members or publicly. Click the share action in the dashboard menu to open the sharing dialog.

**Internal sharing**: grant access to team members who have Kapptivate accounts.

**Public sharing**: generate a publicly shareable link with optional password protection. Public dashboards are read-only and don't require login. Visitors can interact with date ranges, timezones, aggregation, and variables, but cannot modify the underlying dashboard.

View the owner of each dashboard (the creator) in the table. The "Last modified" column shows when the dashboard was last updated with a relative timestamp (e.g., "2 hours ago").

## Locking dashboards

Lock a dashboard to prevent accidental edits. Locked dashboards show a lock icon in the table. Unlock a dashboard from the actions menu when you need to make changes.

## Deleting and trash

Delete a dashboard using the actions menu. Deleted dashboards move to the trash collection and can be restored or permanently deleted. Access the trash by navigating to the trash collection if you need to recover a dashboard.

## Sharing and collaboration

Use the dashboard table to quickly see who owns each dashboard and when it was last modified. Favorite frequently-used dashboards for quick access. Lock dashboards to prevent accidental edits, and move deleted dashboards to trash for recovery.
