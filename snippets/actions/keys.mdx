Press keyboard keys individually or in combination. Supports standard keys (Enter, Tab, Escape), special keys (ArrowUp, ArrowDown, Backspace, Delete), and key chords with modifiers (Shift, Control, Alt, Meta/Cmd).

## Channels

- ✅ Web
- ✅ Smartphone (Android/iOS)
- ✅ Cellular

All channels support keyboard input equally.

## Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| keys | Array[String] | Yes | Array of key names to press sequentially. Supports combinations like `Control+a` or `Meta+c`. |

Supported keys include:

| Category | Keys |
|----------|------|
| **Standard** | Enter, Tab, Escape, Backspace, Delete, Space |
| **Navigation** | ArrowUp, ArrowDown, ArrowLeft, ArrowRight, Home, End, PageUp, PageDown |
| **Modifiers** | Shift, Control, Alt, Meta (Cmd on Mac) |
| **Function** | F1–F12 |
| **Other** | Insert, Pause, CapsLock, NumLock, PrintScreen |

## Example

Submit a form with Enter:

```json
{
  "type": "keys",
  "version": "v1.0.0",
  "inputs": {
    "keys": ["Enter"]
  }
}
```

Select all and delete:

```json
{
  "type": "keys",
  "inputs": {
    "keys": ["Control+a", "Delete"]
  }
}
```

Copy text (Mac):

```json
{
  "type": "keys",
  "inputs": {
    "keys": ["Meta+c"]
  }
}
```

## Notes

- Keys are pressed in order; each key/chord is a separate array element.
- Modifiers (Shift, Control, Alt, Meta) can be combined with `+` (e.g., `Shift+Tab` for reverse tab).
- Use after [input](/tests/actions/web/input) to submit forms or navigate.
- Common uses: submit login forms, clear fields, navigate dialogs, dismiss modals.
