`ktm` ships as a single self-contained binary for Linux, macOS, and Windows. Once it is on your PATH, you authenticate with your Kapptivate API key and you are ready to go.

<Note>
  Download links for the current release are being finalized. If you need a build now, ask your Kapptivate contact for the binary for your platform.
</Note>

## Put ktm on your PATH

After downloading the binary for your platform, make it executable and move it onto your PATH:

```bash
chmod +x ktm
sudo mv ktm /usr/local/bin/ktm
```

Confirm the install:

```bash
ktm --version
```

## Authenticate

`ktm` reads your key from the `KAPPTIVATE_API_KEY` environment variable:

```bash
export KAPPTIVATE_API_KEY="your-key"
ktm status
```

`ktm status` checks connectivity, authentication, and the CLI version in one call. A clean result means you are ready.

<Tip>
  In CI, store the key as a secret and expose it as `KAPPTIVATE_API_KEY`. Never hard-code it in a workflow file.
</Tip>
