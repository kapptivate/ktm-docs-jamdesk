Get variable group detail including variants and variable definitions

## Usage

```bash
ktm variables groups show <id> [flags]
```

## Arguments

| Argument | Required |
| --- | --- |
| `id` | Yes |

## Details

Fetch the full group: metadata, variants[] (with is_default), and the
variables[] inside the group. Single call to inspect the entire group.
