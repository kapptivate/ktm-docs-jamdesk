Reboot an agent or a specific SIM slot

## Usage

```bash
ktm agents reboot [flags]
```

## Flags

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| `--confirm` | boolean | — | Execute the reboot (without this, shows preview) |
| `--slot` | int | — | SIM slot ID (cellular only) |
| `--type` | string | — | Agent type: cellular, ethernet, smartphone (required) |
| `--uuid` | string | — | Agent UUID (required) |

## Examples

```bash
ktm agents reboot --uuid abc-123 --type cellular --confirm
ktm agents reboot --uuid abc-123 --type cellular --slot 3 --confirm
```
