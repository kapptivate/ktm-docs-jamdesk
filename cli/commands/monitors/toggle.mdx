Bulk activate or deactivate monitors

## Usage

```bash
ktm monitors toggle [flags]
```

## Flags

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| `--active` | string | — | true to activate, false to deactivate (required) |
| `--ids` | string | — | Comma-separated monitor IDs to toggle (required) |

## Examples

```bash
ktm monitors toggle --ids 42,43 --active false
ktm monitors toggle --ids 42 --active true
```

<Note>
  Global flags (`--output`, `--debug`, `--host`, …) apply to every command. See the [command reference overview](/cli/commands/overview).
</Note>
