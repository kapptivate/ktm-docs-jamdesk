Deactivate a user account

## Usage

```bash
ktm users deactivate [flags]
```

## Flags

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| `--id` | int | — | User ID to deactivate (from 'users list') |

## Examples

```bash
# Deactivate user with ID 42
ktm users deactivate --id 42
```

## Details

Deactivate a user account by ID. Requires superadmin or EDIT_OPERATOR_ADMIN permission.

Use 'users list' to find user IDs.
