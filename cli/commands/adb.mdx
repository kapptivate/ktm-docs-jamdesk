## Details

Connect Android Studio or the ADB CLI to phones on remote smartphone robots.

The tunnel starts a thin TCP relay on your machine. Your local ADB server
connects to it and traffic is relayed to the phone over a secure WebSocket
tunnel through the platform's infrastructure. The device auto-registers so
Android Studio picks it up automatically.

Each tunnel is scoped to a single phone — you can only see and interact with
the device you connect to.
