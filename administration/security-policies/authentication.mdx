---
title: Configure authentication settings
description: Set up two-factor authentication and login policies.
---

Authentication settings control how users prove their identity when logging in. Configure two-factor authentication and login policies in the collapsible panels within a security policy.

## Two-factor authentication

Expand the **Two Factor Authentication** panel to require 2FA for users assigned to this policy. When enabled, users must provide a second verification factor in addition to their password each time they log in.

## Login policies

Expand the **Login Policy** panel to set rules for how users can log in, such as allowed login methods and IP restrictions.
