---
title: View and manage API keys
description: Search, edit, rotate, and revoke API keys in your workspace.
---

![The row actions menu with Edit and Revoke](/images/api-key-actions.webp)

The API keys list shows every key created in your workspace. Each row shows the key's **Name**, a masked **Token** with an inline rotate icon, and an **Access** column listing how many teams can use the key (or `–` if none). The **⋮** menu at the end of each row opens the **Edit** and **Revoke** actions.

## Search API keys

Use the search box to filter keys by name. Search is case-insensitive and updates in real time.

## Edit an API key

Open the **⋮** menu on a key's row and choose **Edit**. You can change the name and the teams allowed to use the key. Click **Save** to apply your changes.

## Rotate a token

To replace a key's token without changing its name or teams, click the **rotate icon** next to the masked token on the key's row. Kapptivate generates a new token and shows it in a modal. The old token stops working immediately.

Copy the new token before closing the modal. It's only shown once.

## Revoke an API key

Revoking permanently disables a key. Any applications using it will fail to authenticate.

To revoke:

1. Open the **⋮** menu on the key's row and choose **Revoke**
2. Confirm the revocation in the dialog

This action cannot be undone.
