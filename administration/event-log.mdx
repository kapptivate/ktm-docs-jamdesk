The event log records all system events in your workspace -- user logins, policy changes, API key creation, and more. Use it to audit user activity and track changes to your workspace configuration.

<Info>
You must have the appropriate event viewing permission to access this page. If you see a "No Access" message, contact your workspace administrator.
</Info>

## What you can do

- [**Browse and filter events**](/administration/event-log/browse) -- view the event log, filter by date, type, and user, and inspect event details.
- [**Export events**](/administration/event-log/export) -- download filtered event data as a CSV file for offline analysis.

To manage the users whose activity appears in the event log, see [users](/administration/users). To review and update the security policies that generate policy-related events, see [security policies](/administration/security-policies).
