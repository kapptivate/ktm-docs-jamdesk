Web tests drive a real browser against your site, clicking, typing, navigating, and checking what's on the page. They run on a Web agent. Here's the fastest path to your first one.

<Steps>
  <Step>
    **Create the test.** Open the **Tests** section for your product, click **Create**, and choose the **Web** channel. Name the test and set the starting URL the browser opens.
  </Step>
  <Step>
    **Add steps.** Build the flow with web actions, click an element, fill an input, navigate, wait for content, and assert that something is displayed. Save the test.
  </Step>
  <Step>
    **Run it on a browser.** Click **Run** and pick a Web agent. Watch execution live under **Executions**.
  </Step>
  <Step>
    **Read the result.** Open the execution to see status, per-step screenshots, and timing.
  </Step>
</Steps>

<Tip>
Set the browser type, user agent, language, and window size in the test's configuration before you run.
</Tip>
