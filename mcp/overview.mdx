The Kapptivate MCP server exposes test automation, monitoring, and platform data to AI assistants over the Model Context Protocol: letting Claude, Cursor, or any MCP-compatible client read your tests, query metrics, and trigger executions on your behalf.

## Connect

The Kapptivate MCP server is hosted at:

<div className="url-banner" style={{ display: 'flex', alignItems: 'center', gap: '12px', padding: '12px 16px', borderRadius: '8px', fontSize: '14px' }}>
  <i className="fa-light fa-link" style={{ flexShrink: 0 }} />
  <span style={{ fontFamily: 'Geist Mono, monospace' }}>https://mcp.kapptivate.com/mcp</span>
</div>

Authentication uses a personal access token tied to your Kapptivate account.

<Steps>
  <Step title="Get your personal token">

    Sign in to [Kapptivate](https://app.kapptivate.com), open your **Profile** tab, and copy your personal access token.

  </Step>
  <Step title="Configure your MCP client">

    Point your MCP client (Claude Desktop, Cursor, or any MCP-compatible app) at `https://mcp.kapptivate.com/mcp` and provide your personal token in the `Authorization` header as a bearer token.

  </Step>
</Steps>

<div className="url-banner" style={{ display: 'flex', alignItems: 'baseline', gap: '12px', padding: '12px 16px', borderRadius: '8px', fontSize: '14px' }}>
  <i className="fa-light fa-lock" style={{ flexShrink: 0 }} />
  <span>Your personal token grants the same access as your Kapptivate account. Treat it like a password. Never commit it to a repository or share it in plain text.</span>
</div>
