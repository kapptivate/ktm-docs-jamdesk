## Description

The full description the agent receives for this tool:

```text
List the direct sub-collections under a parent collection UID.

Collections are hierarchical containers in the kcollections service that group tests and dashboards. Every product has a tests_root_collection_uid and every operator has a dashboards_root_collection_uid — pass those as parent_uid to browse from the top of each tree.

Returns compact one-liners by default; set detail=true for full JSON (parent_uid, org_id, full UID).

Folder-tree browsing. For flat test discovery by name/tag/status, use list_tests instead — simpler and faster for most queries.
```
