## Description

The full description the agent receives for this tool:

```text
List the reusable test components (kind=2) for a product.

Reusables are V2-only — they work exclusively with step-based tests (ETH_WEB_SCRIPT_V2 / SMARTPHONE_APP_SCRIPT_V2) and are embedded via step_group.import in V2 test definitions. Classic script-based tests cannot use reusables.

Resolves the product's reusable_tests_root_collection_uid, fetches its items via the kcollections service, and returns them with [R] one-liners. (Note: /api/v2/tests/?kind=2 is NOT a reliable path — the backend silently ignores the kind filter. The kcollections route is authoritative.)
```
