Vérifiez ce qu'un selector ne peut pas exprimer : "le graphique montre une tendance à la hausse", "le total du panier correspond à la somme des articles", "aucune bannière d'erreur n'est visible". L'IA observe l'écran courant et juge si votre assertion est vraie.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Assertion | Oui | Ce qui doit être vrai, en langage naturel. Par exemple : "Le panier affiche 2 articles et un total de 39,98 €". Supporte les variables. |
| Ignorer l'erreur sur ce step | Non | L'exécution continue avec le step suivant si celui-ci échoue. Par défaut : désactivé. |

## Résultats et métriques

- **Temps** : début, fin et durée (en secondes) enregistrés dans le résultat du step.
- **Statut** : succès lorsque l'IA juge l'assertion vraie sur l'écran courant, échec dans le cas contraire, avec l'explication de l'IA dans le résultat.
- **Capture d'écran** : prise pour l'état vérifié, afin que vous puissiez examiner ce que l'IA a évalué.
- **Utilisation IA** : le coût IA de l'évaluation est ajouté à l'exécution.

Pour rédiger des assertions fiables, consultez [Vérifier avec l'IA](/fr/ai/verify-with-ai).
