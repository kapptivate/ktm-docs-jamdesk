Détectez les fautes de frappe et les erreurs de grammaire dans votre interface avant vos utilisateurs. L'IA lit le texte visible de la page courante et signale tout ce qui semble incorrect dans la langue que vous spécifiez. Utile sur les pages riches en contenu et pour les vérifications de localisation.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Langue | Oui | La langue dans laquelle le texte visible est censé être rédigé (par exemple `French`). |
| Ignorer l'erreur sur ce step | Non | L'exécution continue avec le step suivant si celui-ci échoue. Par défaut : désactivé. |

## Résultats et métriques

- **Temps** : début, fin et durée (en secondes) enregistrés dans le résultat du step.
- **Statut** : échec lorsque l'IA détecte des fautes d'orthographe ou de grammaire dans le texte visible, avec la liste des problèmes dans le résultat.
- **Capture d'écran** : prise pour l'état vérifié, afin que vous puissiez voir exactement ce qui a été contrôlé.
- **Utilisation IA** : le coût IA de la vérification est ajouté à l'exécution.

Pour comprendre le fonctionnement de la vérification et interpréter les résultats, consultez [Vérification orthographique](/fr/ai/spell-check).
