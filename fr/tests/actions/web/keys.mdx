Envoyez une ou plusieurs touches clavier à la page : Entrée pour soumettre un formulaire, Échap pour fermer une modale, Tab pour déplacer le focus, ou des raccourcis comme Ctrl+A. Dans le builder, vous enregistrez les touches dans une petite modale clavier : appuyez sur les touches que vous souhaitez envoyer, et elles sont stockées dans l'ordre.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Touches | Oui | Les touches à envoyer, enregistrées dans la modale clavier et envoyées dans l'ordre. |
| Ignorer l'erreur sur ce step | Non | L'exécution continue avec le step suivant si celui-ci échoue. Par défaut : désactivé. |

Ce step ne cible pas d'élément : les touches sont envoyées à l'élément qui a actuellement le focus. Utilisez [Click](/fr/tests/actions/web/click) ou [Fill input](/fr/tests/actions/web/input) au préalable pour définir le focus.

## Résultats et métriques

- **Temps d'exécution** : début, fin et durée (en secondes) enregistrés dans le résultat du step.
- **Statut** : succès, avertissement ou échec, avec le message d'erreur en cas d'échec.
- **Capture d'écran** : prise après l'envoi des touches.
