Définissez une valeur une fois et réutilisez-la dans tout le test : un nom d'utilisateur généré au début de l'exécution, une constante que vous ne voulez pas répéter, une valeur construite à partir d'autres variables.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Nom de la variable | Oui | Nom de la variable à créer ou écraser. |
| Valeur | Oui | La valeur à stocker. Supporte le texte, d'autres variables et des fonctions comme le texte aléatoire. |

## Résultats et métriques

- **Variable** : disponible pour chaque step ultérieur du test. Insérez-la partout où un champ supporte les variables.
- **Temps et statut** : enregistrés dans le résultat du step comme pour tout autre step.

Consultez [Variables](/fr/tests/variables) pour les types de variables et les fonctions utilisables dans les valeurs.
