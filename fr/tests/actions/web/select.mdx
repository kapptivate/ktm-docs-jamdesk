<Danger>
  **Obsolète.** Ce step n'apparaît plus dans la palette de steps. Les tests existants qui l'utilisent continuent de fonctionner. Pour les nouveaux tests, utilisez [Click](/fr/tests/actions/web/click) pour ouvrir la liste déroulante et cliquer sur l'option, ce qui fonctionne aussi pour les listes déroulantes personnalisées (non natives).
</Danger>

Choisissez une option dans un élément `<select>` natif, par texte visible, par position ou par attribut.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Élément cible | Oui, sauf si un selector manuel est défini | Description en langage naturel de l'élément select. |
| Selector manuel | Non | Selector XPath ou CSS, utilisé lorsque l'IA est désactivée ou comme selector enregistré. |
| Sélectionner l'option par | Oui | `visible_text` (par défaut), `index`, ou `attribute`. |
| Valeur | Quand by est `visible_text` | Le texte visible de l'option. |
| Index | Quand by est `index` | La position de l'option, à partir de 0. |
| Nom et valeur de l'attribut | Quand by est `attribute` | L'attribut HTML à faire correspondre, par exemple `value` = `FR`. |
| Utilisation IA | Non | `inherit` (par défaut, suit la stratégie du test), `smart`, ou `always`. |
| Ignorer l'erreur sur ce step | Non | L'exécution continue avec le step suivant si celui-ci échoue. Par défaut : désactivé. |

## Résultats et métriques

- **Temps** : début, fin et durée (en secondes) enregistrés dans le résultat du step.
- **Statut** : échec si l'élément ou l'option demandée ne peut pas être trouvé.
- **Capture d'écran** : prise après la sélection.
