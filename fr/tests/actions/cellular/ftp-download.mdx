title: FTP Download
description: Téléchargez un fichier via FTP et mesurez le débit.
---

Téléchargez un fichier depuis un serveur FTP via la connexion data de la SIM et mesurez le débit. Utile quand votre infrastructure de benchmark repose sur FTP ou pour comparer les protocoles sur le même réseau.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Lien de téléchargement | Oui | URL FTP du fichier. |
| Limite de taille de téléchargement | Non | Arrêter après ce nombre d'octets. |
| Identifiants FTP | Non | Login pour le serveur FTP. |
| Le téléchargement doit échouer | Non | Inverse le verdict. |

Ainsi que les [paramètres de connexion](/fr/tests/actions/cellular/overview#paramètres-de-connexion) partagés et la surcharge DNS (`command.override_dns`, `command.dns`).

## Résultats et métriques

Dans le rapport de l'action (`result`) : `throughput_in_kbps`, `md5sum` (intégrité du fichier), `dns_server` et `duration` en secondes.

Dans Analytics, chaque exécution alimente la mesure FTP download avec `throughput_kbps` et `duration_seconds`, étiquetée par statut, URL, APN, antenne et technologie d'accès.
