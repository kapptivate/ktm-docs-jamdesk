---
title: Send Mail
description: Envoyez un e-mail via SMTP sur la connexion cellulaire.
---

Envoyez un e-mail via un serveur SMTP sur la connexion data de la SIM. Vérifiez que l'envoi d'e-mails fonctionne depuis le réseau mobile, y compris à travers le NAT opérateur et le filtrage SMTP.

## Paramètres

| Paramètre | Requis | Description |
| ----------- | -------- | ------------- |
| Serveur SMTP | Oui | L'adresse du serveur SMTP. |
| Compte SMTP | Oui | Identifiants utilisés pour l'envoi. |
| Destinataire | Oui | Adresse e-mail de destination. |
| Objet | Non | Objet de l'e-mail. |
| Corps | Non | Contenu de l'e-mail. |
| L'envoi doit échouer | Non | Inverse le verdict, pour le SMTP qui doit être bloqué. |

Ainsi que les [paramètres de connexion](/fr/tests/actions/cellular/overview#paramètres-de-connexion) partagés.

## Résultats et métriques

Dans le rapport de l'action (`result`) : le `status` d'envoi et la `duration` en secondes.

Dans Analytics, chaque exécution alimente la mesure d'envoi d'e-mail avec `duration_seconds`, étiquetée par statut, APN, antenne et technologie d'accès.
