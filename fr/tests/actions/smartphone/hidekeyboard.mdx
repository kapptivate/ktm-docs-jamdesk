Masquez le clavier virtuel après la saisie. Le clavier peut recouvrir des boutons et des champs, alors masquez-le avant d'appuyer sur quoi que ce soit dans la moitié inférieure de l'écran.

## Paramètres

Ce step n'a pas de paramètres.

## Dans le JSON du scénario

Le step se trouve dans `step_groups[].steps` du scénario de test.

```json
{
  "id": "step-1",
  "type": "hide-keyboard",
  "version": "v1.0.0",
  "disabled": false,
  "inputs": {},
  "use_capabilities": false,
  "capabilities": {}
}
```

## Résultats et métriques

- **Timing** : début, fin et durée (en secondes) enregistrés dans le résultat du step.
- **Statut** : succès, même si aucun clavier n'était ouvert.
- **Capture d'écran** : prise après la fermeture du clavier.
