Exportez votre journal des événements en CSV pour une analyse hors ligne, des rapports ou de la documentation de conformité.

## Télécharger une exportation CSV

Cliquez sur le bouton **Exporter CSV** (en haut à droite) pour télécharger les événements filtrés sous forme de fichier CSV. L'exportation inclut :

- Tous les événements correspondant à vos filtres actuels (plage de dates, type, utilisateur)
- Toutes les colonnes affichées dans le tableau
- Les horodatages dans le fuseau horaire sélectionné

Vous pouvez importer le CSV dans des outils de tableur ou de business intelligence pour une analyse approfondie.

<Tip>
Appliquez des filtres avant d'exporter pour limiter le fichier aux événements dont vous avez besoin. Une exportation sans filtre inclut tous les événements de votre workspace.
</Tip>
