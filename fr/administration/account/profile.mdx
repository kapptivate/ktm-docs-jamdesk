Votre profil est votre identité dans le workspace : votre nom, votre nom d'utilisateur et votre email, ainsi que vos appartenances (équipes), vos accès (produits) et les identifiants liés à votre compte. Tout se trouve sur une seule page, et le menu d'ancrage sur la gauche permet de naviguer entre les sections.

Pour mettre à jour votre identité :

1. Cliquez sur le nom de votre workspace en haut de la barre latérale et sélectionnez **Settings**
2. Ouvrez **Profile** sous My account
3. Modifiez votre prénom, votre nom ou votre nom d'utilisateur (votre email est en lecture seule)
4. Cliquez sur **Save changes** (en haut à droite)

Votre nom d'utilisateur doit être unique. Le système vérifie la disponibilité pendant la saisie, avec une courte pause avant la validation pour éviter les vérifications redondantes.

## Équipes et accès aux produits

Deux sections affichent vos accès et vous permettent de les étendre :

- **User Teams** : les équipes auxquelles vous appartenez. Cliquez sur **+ Add team** pour en rejoindre une.
- **Product Access** : les produits sur lesquels vous pouvez travailler. Cliquez sur **+ Add product** pour en ajouter un.

## Politique de sécurité

La section **Security & Policy** affiche la [politique de sécurité](/fr/administration/security-policies) appliquée à votre compte, avec un menu déroulant pour en sélectionner une. La politique par défaut du workspace s'applique sauf si une autre est sélectionnée.

## Jeton d'accès personnel

La section **Personal access token** contient un identifiant lié à votre compte qui authentifie les outils externes comme le [CLI Kapptivate](/fr/cli/overview) et le [serveur MCP](/fr/mcp/overview). Traitez-le comme un mot de passe.

Pour l'utiliser, cliquez sur l'icône en forme d'œil pour révéler le jeton, puis sur l'icône de copie pour le copier dans l'outil que vous configurez. Le jeton est régénéré lorsque vous vous déconnectez, vous devrez donc le copier à nouveau après votre prochaine connexion.
