Activer ou désactiver plusieurs monitors en masse.

<Warning>
  **Destructif · Écriture de données · Idempotent · Appelle des systèmes externes.** Cet outil peut modifier ou supprimer des données. Vérifiez les arguments avant de laisser un agent l'appeler.
</Warning>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `active` | boolean | Oui | true pour activer, false pour désactiver |
| `monitor_ids` | string | Oui | Identifiants de monitors séparés par des virgules |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "monitor_ids": "string",
  "active": false
}
```

## Description

```text
Utilisez list_monitors pour trouver les identifiants. Si l'opération échoue avec HTTP 400, un ou plusieurs identifiants de monitors peuvent être invalides ou supprimés — vérifiez avec get_monitor. Pour un seul monitor, edit_monitor active= fonctionne aussi ; ceci est le chemin pour les opérations en masse.
```
