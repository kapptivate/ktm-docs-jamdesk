<Note>
  **Lecture seule · Appelle des systèmes externes.** Appel sûr : cet outil ne modifie pas les données.
</Note>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `variable_id` | number | Oui | Identifiant de la variable |
| `with_usage` | boolean | Non | Si true, enrichit la réponse avec la liste des tests utilisant cette variable. False par défaut. |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "variable_id": 0
}
```

## Description

```text
Les secrets restent masqués.

Compact par défaut, renvoie uniquement la variable. Définissez with_usage=true pour inclure également la liste des tests qui la référencent ; la réponse est alors structurée en {variable, usage}.
```
