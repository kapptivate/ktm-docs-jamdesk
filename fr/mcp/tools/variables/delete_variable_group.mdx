Supprimer un groupe de variables.

<Warning>
  **Destructif · Écriture de données · Idempotent · Appelle des systèmes externes.** Cet outil peut modifier ou supprimer des données. Vérifiez les arguments avant de laisser un agent l'appeler.

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `acknowledge` | boolean | Oui | Doit être true pour confirmer la suppression |
| `force` | boolean | Non | Contourner la vérification de non-vacuité (false par défaut) |
| `group_id` | number | Oui | Identifiant du groupe de variables |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "group_id": 0,
  "acknowledge": false
}
```

## Description

```text
Sécurisé par défaut : refuse si le groupe contient encore des variables, en les listant. acknowledge=true est requis. force=true contourne la vérification locale de non-vacuité ; le refus propre à la plateforme s'applique toujours.
```
