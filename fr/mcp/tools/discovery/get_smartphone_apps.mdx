Liste les applications installables sur un smartphone pour la création de tests smartphone V2/classiques. Copiez `package` vers `starting.app`, l'app vers `meta`.

<Note>
  **Lecture seule · Appelle des systèmes externes.** Appel sûr : cet outil ne modifie pas les données.
</Note>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `detail` | boolean | Non | Renvoie les objets app complets \{kind,name,label,package,version\} au lieu de lignes compactes (défaut : false) |
| `filter` | string | Non | Filtre insensible à la casse sur le nom du package ou de l'app |
| `serial` | string | Oui | UUID/serial du device (depuis list_resources type=smartphone) |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "serial": "string"
}
```

## Description

```text
Find serials via `list_resources type=smartphone`. Compact by default; use filter= to narrow, detail=true for full objects.
```
