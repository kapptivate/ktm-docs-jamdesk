<Note>
  **Lecture seule · Appelle des systèmes externes.** Appel sûr : cet outil ne modifie pas les données.
</Note>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `detail` | boolean | Non | Renvoie le JSON complet au lieu de résumés compacts (défaut : false) |
| `organisation_group` | string | Non | Filtrer par groupe d'organisation |
| `short_name` | string | Non | Filtrer par nom court de l'operator |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json

  "detail": false,
  "organisation_group": "string"
}
```

## Description

```text
Returns compact summaries by default; use detail=true for full JSON. List operators accessible with the current API key. Operators are the top-level organisational unit in Kapptivate.
```
