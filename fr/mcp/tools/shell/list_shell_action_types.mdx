## Description

```text
Les actions shell sont un sous-ensemble organisé : uniquement les opérations directes sur appareil/réseau (USSD, ping, HTTP API, etc.). Les actions d'exécution de scripts et de contrôle de tests sont exclues. Utilisez cet outil avant send_shell_command pour voir les types disponibles et leurs champs requis. Pour l'historique des commandes déjà envoyées, utilisez list_shell_actions.
```
