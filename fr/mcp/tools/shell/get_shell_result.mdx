## Description

```text
À utiliser après send_shell_command pour vérifier le statut et récupérer la sortie. Interrogez régulièrement jusqu'à ce que le statut soit terminal (Done/Timeout/Blocked/Canceled/Maintenance). Pour USSD : vérifiez si la session est OPEN ou CLOSED. Si OPEN, envoyez une entrée de suivi avec send_shell_command pour naviguer dans le menu.
```
