Recherche parmi les collections et éléments (tests + dashboards) sous un sous-arbre de collections.

<Note>
  **Lecture seule · Appelle des systèmes externes.** Appel sûr : cet outil ne modifie pas les données.
</Note>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `collection_uid` | string | Oui | Scope la recherche à ce sous-arbre de collections (typiquement le tests_root_collection_uid d'un produit) |
| `detail` | boolean | Non | Renvoie le JSON complet au lieu de lignes compactes (défaut : false) |
| `operator` | string | Oui | Slug de l'operator (fournit l'org-id) |
| `owner_id` | number | Non | Filtrer par identifiant d'utilisateur propriétaire |
| `query` | string | Non | Recherche texte libre (correspond au nom et aux métadonnées) |
| `tags` | string | Non | Identifiants de tags séparés par des virgules |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json

  "operator": "string",
  "collection_uid": "string"
}
```

## Description

```text
The collections service requires a collection_uid to scope the search — pass the product's tests_root_collection_uid or any sub-collection UID. Returns a payload with two arrays: collections and items.
```
