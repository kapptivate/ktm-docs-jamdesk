Déplace une collection dans la corbeille de l'organisation.

<Warning>
  **Destructif · Écrit des données · Idempotent · Appelle des systèmes externes.** Cet outil peut modifier ou supprimer des données. Vérifiez les arguments avant de laisser un agent l'appeler.
</Warning>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `acknowledge` | boolean | Oui | Doit être true : confirme que l'appelant accepte de déplacer cette collection dans la corbeille (suppression réversible, récupérable tant qu'un admin n'a pas vidé la corbeille) |
| `operator` | string | Oui | Slug de l'operator |
| `uid` | string | Oui | UID de la collection à supprimer |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "operator": "string",
  "uid": "string",
  "acknowledge": false
}
```

## Description

```text
The collection is recoverable until an admin empties the trash on the collections service; there is no direct hard-delete. Refuses unless acknowledge=true. Recommended flow: list_collection_items (review the contents) → delete_collection with acknowledge=true.
```
