Activer ou désactiver le mode maintenance d'un agent.

<Warning>
  **Destructif · Écriture de données · Idempotent · Appelle des systèmes externes.** Cet outil peut modifier ou supprimer des données. Vérifiez les arguments avant de laisser un agent l'appeler.
</Warning>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `confirm` | boolean | Oui | Doit être true pour exécuter. Sans confirm=true, renvoie un aperçu. |
| `maintenance` | boolean | Oui | true pour activer la maintenance, false pour la désactiver |
| `uuid` | string | Oui | UUID de l'agent |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "uuid": "string",
  "maintenance": false,
  "confirm": false
}
```

## Description

```text
Fonctionne pour tous les types d'agents.
Nécessite confirm=true pour exécuter. Sans cela, renvoie un aperçu du changement.

Le mode maintenance empêche l'agent d'exécuter des tests. Utilisez list_agents pour trouver les UUID d'agents.
```
