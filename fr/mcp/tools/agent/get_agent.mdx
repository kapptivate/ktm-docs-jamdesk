Obtenir des informations détaillées sur un agent unique, incluant les slots (cellular/smartphone) ou la capacité (ethernet).

<Note>
  **Lecture seule · Appelle des systèmes externes.** Appel sûr : cet outil ne modifie pas les données.
</Note>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `type` | string (cellular \| ethernet \| smartphone) | Oui | Type d'agent : cellular, ethernet, smartphone |
| `uuid` | string | Oui | UUID de l'agent |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "uuid": "string",
  "type": "cellular"
}
```

## Description

```text
Les agents cellular affichent les slots de carte SIM avec IMEI et le statut de SIM attachée.
Les agents smartphone affichent les téléphones connectés avec les chemins USB et le statut.
Les agents ethernet affichent la capacité (tests parallèles maximum).

Utilisez list_agents pour trouver les UUID d'agents au préalable.
```
