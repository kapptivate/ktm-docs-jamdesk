| Champ | Type | Requis | Description |

## Description

```text
Les agents sont le matériel/logiciel qui exécute les ressources, limités à un opérateur et non à un produit.
Trois types : cellular (passerelles GSM avec slots SIM), ethernet (web agents autonomes), smartphone (hubs téléphoniques).

Renvoie un en-tête de résumé de capacité montrant la capacité totale en ligne pour les tests simultanés.
Utilisez get_agent pour le détail avec les slots. Utilisez list_resources pour les informations sur les appareils par produit.
```
