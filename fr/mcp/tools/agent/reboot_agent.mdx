Redémarrer un agent ou un slot SIM spécifique (cellular uniquement).

<Warning>
  **Destructif · Écriture de données · Appelle des systèmes externes.** Cet outil peut modifier ou supprimer des données. Vérifiez les arguments avant de laisser un agent l'appeler.
</Warning>

## Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| `confirm` | boolean | Oui | Doit être true pour exécuter. Sans confirm=true, renvoie un aperçu. |
| `slot_id` | number | Non | Identifiant du slot SIM spécifique à redémarrer (cellular uniquement, depuis les slots de get_agent) |
| `type` | string (cellular \| ethernet \| smartphone) | Oui | Type d'agent : cellular, ethernet, smartphone |
| `uuid` | string | Oui | UUID de l'agent |

## Exemple d'arguments

Arguments illustratifs qu'un agent fournit lors de l'appel de cet outil :

```json
{
  "uuid": "string",
  "type": "cellular",
  "confirm": false
}
```

## Description

```text
Nécessite confirm=true pour exécuter. Sans cela, renvoie un aperçu de ce qui sera redémarré.

Utilisez list_agents pour trouver les UUID d'agents. Utilisez get_agent pour voir les identifiants de slots pour un redémarrage au niveau du slot.
```
