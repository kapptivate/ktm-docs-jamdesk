## Guide pas à pas

## Aperçu

L'onglet **Aperçu** affiche les dernières exécutions du test. Cliquez sur une exécution pour voir le détail de l'étape en échec.

## Environnement

Dans l'onglet **Environnement**, vous pouvez :

- Modifier l'application utilisée pour le test
- Changer la ressource (téléphone) par défaut

## Paramètres du test

L'onglet **Paramètres de test** regroupe les réglages généraux.

<Steps>
  <Step title="Description et tags">
    Ajoutez une description pour documenter le test et des tags pour faciliter le tri.
  </Step>
  <Step title="Paramètres IA">
    Cliquez sur **IA** pour modifier les paramètres d'intelligence artificielle. L'option **Smart** est recommandée.
  </Step>
  <Step title="Délais et timeouts">
    Cliquez sur **Délais et timeouts** pour ajuster le délai entre chaque étape et les durées maximales d'attente.
  </Step>
</Steps>

## Hooks

Les hooks exécutent des actions avant ou après le test.

<Steps>
  <Step title="Configurer un hook">
    Dans la section **Hooks**, cliquez sur **Before test** (ou After test).
  </Step>
  <Step title="Ajouter un composant réutilisable">
    Cliquez sur **Ajouter** et sélectionnez le composant réutilisable à exécuter.
  </Step>
</Steps>

## Options avancées

Cliquez sur l'onglet **Avancé** pour accéder aux paramètres suivants :

- **Réinitialiser l'application** : remet l'application dans son état initial avant chaque exécution

## Historique des versions

<Steps>
  <Step title="Consulter l'historique">
    Accédez à l'onglet **Historique des versions** pour voir les modifications passées.
  </Step>
  <Step title="Restaurer une version">
    Sélectionnez la version souhaitée, cliquez sur l'icône d'action et choisissez **Restaurer cette version**.
  </Step>
</Steps>
