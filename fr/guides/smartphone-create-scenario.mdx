## Guide pas à pas

## Créer le test

<Steps>
  <Step title="Ouvrir la liste des tests">
    Dans le menu principal, cliquez sur **Tests**.
  </Step>
  <Step title="Lancer la création">
    Cliquez sur **Créer un nouveau test**, puis sélectionnez **Smartphone (Nouvelle expérience)**.
  </Step>
  <Step title="Nommer le test">
    Donnez un nom descriptif à votre test.
  </Step>
  <Step title="Choisir le téléphone">
    Sélectionnez le smartphone à utiliser. Assurez-vous qu'il n'est pas utilisé par un autre test ou un monitoring actif.
  </Step>
  <Step title="Choisir l'application">
    Sélectionnez l'application à tester, puis cliquez sur **Créer**.
  </Step>
</Steps>

## Enregistrer les étapes

<Steps>
  <Step title="Démarrer le recorder">
    Cliquez sur **Utiliser le recorder**. L'application se lance sur le téléphone. Chaque action que vous effectuez est enregistrée automatiquement.
  </Step>
  <Step title="Effectuer le parcours">
    Interagissez avec l'application : tapez sur les boutons, remplissez les champs, naviguez entre les écrans. Les étapes apparaissent en temps réel dans le panneau de gauche.
  </Step>
  <Step title="Ajouter une assertion">
    Cliquez sur l'icône **Mode assertion**, puis tapez sur l'élément à vérifier (par exemple un message d'erreur). Cela ajoute une étape de vérification au scénario.
  </Step>
  <Step title="Arrêter l'enregistrement">
    Cliquez sur **Arrêter l'enregistrement**.
  </Step>
</Steps>

## Exécuter et vérifier

<Steps>
  <Step title="Enregistrer et exécuter">
    Cliquez sur **Enregistrer et exécuter** pour sauvegarder le scénario et le lancer immédiatement.
  </Step>
  <Step title="Consulter les résultats">
    Vérifiez le statut de chaque étape. En cas d'échec, identifiez l'étape concernée pour corriger le scénario.
  </Step>
</Steps>
