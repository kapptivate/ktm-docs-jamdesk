## Guide pas à pas

<Steps>
  <Step title="Ouvrir une session live">
    Depuis le portail Kapptivate, cliquez sur **Live session**, puis sélectionnez **Web** pour accéder à l'agent web.
  </Step>
  <Step title="Saisir l'URL de l'API">
    Dans le champ "Enter the request URL", saisissez l'URL de votre endpoint API.
  </Step>
  <Step title="Exécuter manuellement">
    Cliquez sur le bouton vert pour lancer l'appel API et vérifier la réponse.
  </Step>
  <Step title="Consulter l'historique">
    Cliquez sur **History** pour voir les appels précédents.
  </Step>
  <Step title="Ajouter au scénario">
    Ajoutez l'appel API à votre liste d'étapes de test.
  </Step>
  <Step title="Créer le test">
    Cliquez sur **Créer le test**, donnez-lui un nom descriptif.
  </Step>
  <Step title="Enregistrer et exécuter">
    Cliquez sur **Enregistrer et exécuter** pour valider que le scénario fonctionne.
  </Step>
  <Step title="Vérifier les résultats">
    Consultez le statut de l'exécution pour confirmer que le test est opérationnel.
  </Step>
</Steps>
