## Guide pas à pas

## Créer le test

<Steps>
  <Step title="Ouvrir la liste des tests">
    Dans le menu principal, cliquez sur **Tests**.
  </Step>
  <Step title="Sélectionner le produit">
    Vérifiez que le bon produit est sélectionné dans le sélecteur en haut de la page.
  </Step>
  <Step title="Lancer la création">
    Cliquez sur **Créer un nouveau test**, puis sélectionnez **Web (Nouvelle expérience)**.
  </Step>
  <Step title="Définir l'URL de départ">
    Saisissez l'URL de la page sur laquelle le test doit commencer, puis cliquez sur **Créer**.
  </Step>
</Steps>

## Enregistrer les étapes

<Steps>
  <Step title="Démarrer le recorder">
    Cliquez sur **Utiliser le recorder**, puis confirmez avec **OK**. Chaque action que vous effectuez dans le navigateur est enregistrée automatiquement.
  </Step>
  <Step title="Effectuer le parcours">
    Naviguez sur le site comme le ferait un utilisateur : cliquez sur les boutons, remplissez les champs, suivez les liens.
  </Step>
  <Step title="Ajouter des assertions">
    Pour vérifier qu'un élément est bien affiché, sélectionnez l'option **Affiché** dans le recorder. Cela ajoute une étape de vérification au scénario.
  </Step>
  <Step title="Terminer l'enregistrement">
    Faites un clic droit pour arrêter le recorder, puis cliquez sur **Enregistrer** pour sauvegarder les étapes.
  </Step>
</Steps>

## Lancer l'exécution

<Steps>
  <Step title="Exécuter le test">
    Cliquez sur **Exécuter**. Le test se lance sur la ressource sélectionnée.
  </Step>
  <Step title="Consulter les résultats">
    Une fois l'exécution terminée, cliquez sur l'identifiant de l'exécution pour accéder aux détails. Chaque étape affiche son statut (succès ou échec) et une capture d'écran.
  </Step>
</Steps>
