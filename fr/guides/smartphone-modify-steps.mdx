## Guide pas à pas

## Modifier le type d'action

<Steps>
  <Step title="Ouvrir le sélecteur d'action">
    Cliquez sur le type d'action d'une étape existante (par exemple "Clic").
  </Step>
  <Step title="Choisir le nouveau type">
    Sélectionnez le type souhaité dans la liste. Par exemple, remplacez un **Clic** par un **Appui long**.
  </Step>
</Steps>

## Ajouter des étapes

<Steps>
  <Step title="Masquer le clavier">
    Ajoutez une étape et choisissez **Masquer le clavier** si le clavier gêne votre parcours.
  </Step>
  <Step title="Attendre un délai">
    Ajoutez une étape **Attendre un délai** et définissez la durée si votre scénario nécessite une pause.
  </Step>
</Steps>

## Récupérer des emails et codes

<Steps>
  <Step title="Vérifier la réception d'un email">
    Ajoutez une étape **Obtenir un e-mail** et saisissez l'adresse de réception (doit se terminer par `@kappti.dev`).
  </Step>
  <Step title="Récupérer un code OTP par email">
    Ajoutez une étape **Obtenir le code e-mail** avec l'adresse `@kappti.dev`. Le code est stocké dans la variable `MailCode` que vous pouvez réutiliser plus tard.
  </Step>
  <Step title="Récupérer un lien par email">
    Ajoutez une étape **Obtenir le lien e-mail** avec l'adresse `@kappti.dev`. Le lien est stocké dans la variable `MailLink`.
  </Step>
</Steps>

<Note>
Toutes les adresses email utilisées pour la récupération doivent se terminer par `@kappti.dev`.
</Note>

## Ajouter des vérifications

<Steps>
  <Step title="Ajouter une vérification d'élément">
    Ajoutez une étape **Vérifier un élément**.
  </Step>
  <Step title="Configurer la condition">
    Choisissez si l'élément **doit** ou **ne doit pas** correspondre à la condition.
  </Step>
  <Step title="Choisir le type de vérification">
    Sélectionnez parmi : **Affiché**, **Cliquable**, **Activé** ou **Présent**.
  </Step>
</Steps>
