Obtenir les détails d'une définition d'alerte

## Utilisation

```bash
ktm alerts get [flags]
```

## Exemples

```bash
# Get alert definition details
ktm alerts get --id 42
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--id` | int | — | ID de la définition d'alerte (depuis 'alerts list') |

## Détails

Obtenir une seule définition d'alerte par ID, incluant sa configuration et ses paramètres de notification.

Utilisez 'alerts list' pour trouver les ID des définitions d'alertes.
