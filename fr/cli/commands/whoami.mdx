Afficher l'utilisateur actuellement connecté (depuis le token OAuth stocké)

## Utilisation

```bash
ktm whoami [flags]
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--creds-file` | string | — | Chemin vers le fichier d'identifiants de secours |
| `--no-keychain` | boolean | — | Opère sur le fichier de secours plutôt que sur le trousseau de clés OS |

## Détails

Affiche l'identité portée par le token OAuth stocké : sujet (ID utilisateur),
e-mail, portées accordées, émetteur et durée avant expiration.

Il s'agit d'une lecture locale uniquement ; elle ne fait PAS appel au serveur d'autorisation.
Les informations affichées sont lues directement depuis le payload JWT (affichage
non vérifié uniquement).

</Note>
