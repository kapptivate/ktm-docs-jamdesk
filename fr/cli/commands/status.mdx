Vérifier la connectivité, l'authentification et la version du CLI

## Utilisation

```bash
ktm status [flags]
```

## Exemples

```bash
ktm status
```

## Détails

Vérification rapide de l'environnement CLI.

Affiche l'hôte configuré, l'état d'authentification, la version du CLI et la latence de l'API.
Utile pour vérifier la configuration avant d'exécuter des commandes.

<Note>
  Les flags globaux (`--output`, `--debug`, `--host`, ...) s'appliquent à toutes les commandes. Consultez la [vue d'ensemble de la référence des commandes](/fr/cli/commands/overview).
</Note>
