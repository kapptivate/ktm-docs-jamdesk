Obtenir les résultats détaillés d'une exécution de test

## Utilisation

```bash
ktm executions get <id> [flags]
```

## Exemples

```bash
# Get execution detail as JSON
ktm executions get 12345

# Get execution detail as table
ktm executions get 12345 -o table
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Récupère une seule exécution par ID et affiche ses résultats détaillés.

La réponse inclut :
  - Métadonnées d'exécution : nom, statut, nombre de réussites/échecs, créateur, horodatages
  - tests_overview : résumé rapide réussite/échec par test
  - json : données d'exécution complètes avec actions par test, timing, erreurs et variables

En mode table, un résumé lisible est affiché avec un lien vers l'interface Kapptivate.
En mode JSON, la réponse complète est retournée incluant le champ json brut.
