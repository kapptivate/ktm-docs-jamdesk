Ouvrir un tunnel ADB vers un téléphone distant

## Utilisation

```bash
ktm adb connect <phone-serial> [flags]
```

## Exemples

```bash
# Connect to a phone (auto-registers with local ADB server)
ktm adb connect RF8NB07VDGA

# Custom port
ktm adb connect RF8NB07VDGA --port 7037
```

## Arguments

| Argument | Requis |
| --- | --- |
| `phone-serial` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--port` | int | `6037` | Port local d'écoute |

## Détails

Démarre un tunnel ADB local vers un téléphone spécifique sur un robot smartphone distant.

Un relais TCP léger fait le pont entre votre serveur ADB local et le téléphone distant via un
tunnel WebSocket sécurisé. L'appareil s'enregistre automatiquement auprès de votre serveur ADB local,
pour qu'Android Studio le détecte. Support complet du port forwarding,
du reverse forwarding et du débogage JDWP : votre vrai serveur ADB local gère
toutes les questions de protocole.
