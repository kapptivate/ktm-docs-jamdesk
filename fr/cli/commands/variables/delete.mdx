Supprimer une variable ; refuse si un test la référence

## Utilisation

```bash
ktm variables delete <id> [flags]
```

## Exemples

```bash
ktm variables delete 17
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Refuse si un test référence la variable, en listant les références. Le
endpoint DELETE de la plateforme applique la même protection côté serveur et n'expose aucun
mécanisme de contournement, donc il n'y a pas de flag --force ici : supprimez d'abord les références via
'ktm tests edit' (ou supprimez les tests concernés) puis relancez la suppression.
