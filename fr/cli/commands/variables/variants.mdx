## Détails

Gérer les variants sous un groupe de variables : ajouter, supprimer (sécurisé par défaut),
renommer, définir le défaut et copier des valeurs entre variants. Utilisez
'ktm variables groups show &lt;id&gt;' pour lister les variants actuels.
