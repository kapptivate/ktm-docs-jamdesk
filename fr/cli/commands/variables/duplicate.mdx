Cloner une variable sous un nouveau nom (atomique ; conserve le type et les valeurs)

## Utilisation

```bash
ktm variables duplicate <id> [flags]
```

## Exemples

```bash
ktm variables duplicate 17 --name DB_HOST_BACKUP
```

| Argument | Requis |

| `id` | Oui |

## Flags

| --- | --- | --- | --- |
| `--name` | string | — | Nom pour la nouvelle variable (requis) |

## Détails

Le duplicata hérite du type, du groupe et de toutes les valeurs par variant de la source.
