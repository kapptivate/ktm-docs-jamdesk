Obtenir les détails d'un groupe de variables incluant les variants et définitions de variables

## Utilisation

```bash
ktm variables groups show <id> [flags]
```

## Exemples

```bash
ktm variables groups show 204
ktm variables groups show 204 -o table
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Récupérer le groupe complet : métadonnées, variants[] (avec is_default) et les
variables[] du groupe. Un seul appel pour inspecter l'ensemble du groupe.
