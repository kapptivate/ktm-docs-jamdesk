Supprimer un groupe de variables ; refuse s'il n'est pas vide sauf avec --force

## Utilisation

```bash
ktm variables groups delete <id> [flags]
```

## Exemples

```bash
ktm variables groups delete 42           # refuse si le groupe contient des variables
ktm variables groups delete 42 --force   # contourne la vérification de non-vacuité
```

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--force` | boolean | — | Contourner la vérification de sécurité de non-vacuité |

## Détails

Sécurisé par défaut : refuse si le groupe contient encore des variables, en les listant
et en suggérant --force pour passer outre.
