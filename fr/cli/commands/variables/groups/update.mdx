Renommer un groupe de variables (la gestion des variants est dans 'variables variants')

## Utilisation

```bash
ktm variables groups update <id> [flags]
```

## Exemples

```bash
ktm variables groups update 42 --name "Environments v2"
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--name` | string | — | Nouveau nom (requis) |

## Détails

Renommage uniquement. Pour ajouter, supprimer, renommer, définir le défaut ou copier des valeurs entre
variants, utilisez les sous-commandes dédiées 'ktm variables variants'.
