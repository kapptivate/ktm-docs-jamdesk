Ajouter un nouveau variant à un groupe de variables

## Utilisation

```bash
ktm variables variants add <group-id> [flags]
```

## Exemples

```bash
ktm variables variants add 42 --name canary
ktm variables variants add 42 --name canary --default
```

## Arguments

| Argument | Requis |
| --- | --- |
| `group-id` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--default` | boolean | — | Définir le nouveau variant comme défaut (rétrograde l'ancien défaut) |
| `--name` | string | — | Nom du nouveau variant (requis) |

## Détails

Ajoute un nouveau variant au groupe. Avec --default, le nouveau variant devient
le défaut et l'ancien défaut est rétrogradé dans la même mise à jour atomique.
