Promouvoir les valeurs de variants : copier la valeur du variant --from vers --to pour chaque variable du groupe

## Utilisation

```bash
ktm variables variants copy-values <group-id> [flags]
```

## Exemples

```bash
ktm variables variants copy-values 42 --from staging --to prod
ktm variables variants copy-values 42 --from 1 --to 2
```

## Arguments

| Argument | Requis |
| --- | --- |
| `group-id` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--from` | string | — | Identifiant ou nom du variant source (requis) |
| `--to` | string | — | Identifiant ou nom du variant de destination (requis) |

## Détails

Workflow puissant : en une commande, copier la valeur du variant source
vers le variant de destination pour chaque variable de l'ensemble du groupe. --from et --to acceptent
des identifiants ou noms de variants. Les variables qui n'ont pas de valeur sur le variant source sont
ignorées.
