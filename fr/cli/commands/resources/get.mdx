Obtenir les détails d'un smartphone spécifique par numéro de série/UUID

## Utilisation

```bash
ktm resources get <serial> [flags]
```

## Exemples

```bash
# Obtenir les détails d'un smartphone
ktm resources get R92Y70ZDMKH

# Obtenir un smartphone avec les applications installées (pour la création de tests smartphone V2)
ktm resources get R92Y70ZDMKH --include-apps

# Obtenir un smartphone avec les applications au format tableau
ktm resources get R92Y70ZDMKH --include-apps -o table
```

## Arguments

| Argument | Requis |
| --- | --- |
| `serial` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--include-apps` | boolean | — | Récupérer les applications installées sur l'appareil (pour la création de tests smartphone V2 : copier package vers starting.app, objet app vers meta) |

## Détails

Récupérer les détails d'un appareil smartphone par son numéro de série ou UUID.

Utilisez --include-apps pour récupérer également la liste des applications installées sur l'appareil.
La liste des applications installées correspond directement aux champs starting.app et meta du test V2 :
  - copier package   vers starting.app
  - copier l'objet app \{kind, name, label, package, version\} vers meta

Pour trouver les numéros de série des smartphones, utilisez : ktm resources list --type smartphone
