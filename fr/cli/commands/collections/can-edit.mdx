Vérifier si l'utilisateur courant peut modifier une collection

## Utilisation

```bash
ktm collections can-edit <uid> [flags]
```

## Exemples

```bash
ktm collections can-edit --operator my-op <uid>
ktm collections can-edit --operator my-op <uid> -o json | jq .can_edit
```

## Arguments

| Argument | Requis |
| --- | --- |
| `uid` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--operator` | string | — | Slug de l'opérateur (requis) |

## Détails

Vérification de permission. Utile avant d'appeler rename/move/delete pour éviter un
403. Le service retourne un booléen JSON brut que cette commande encapsule en
\{can_edit: true|false\}.
