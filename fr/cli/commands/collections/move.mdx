Déplacer une collection sous un nouveau parent UID

## Utilisation

```bash
ktm collections move <uid> [flags]
```

## Exemples

```bash
ktm collections move --operator my-op <uid> --new-parent <new_parent_uid>
```

## Arguments

| Argument | Requis |
| --- | --- |
| `uid` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--new-parent` | string | — | UID de la collection parent cible (requis) |
| `--operator` | string | — | Slug de l'opérateur (requis) |

## Détails

Déplace la collection (et tout son sous-arbre) sous un nouveau parent.
Les cycles sont rejetés côté serveur. Utilisez 'ktm collections move-items' pour
déplacer un test ou dashboard plutôt qu'une collection entière.
