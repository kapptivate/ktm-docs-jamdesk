Lister les sous-collections sous un UID parent

## Utilisation

```bash
ktm collections list [flags]
```

## Exemples

```bash
ktm collections list --operator my-op --parent <tests_root_uid>
ktm collections list --operator my-op --parent <uid> -o table
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--operator` | string | — | Slug de l'opérateur (requis) |
| `--parent` | string | — | UID de la collection parent (requis) |

## Détails

Liste les enfants directs d'une collection (tableau plat ; ce listing
n'est pas paginé).

Les collections ont une portée opérateur. Commencez par le tests_root_collection_uid d'un produit
(pour l'arborescence de tests) ou le dashboards_root_collection_uid d'un opérateur (pour
l'arborescence de dashboards) et naviguez en profondeur.
