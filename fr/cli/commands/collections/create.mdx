Créer une nouvelle sous-collection sous --parent

## Utilisation

```bash
ktm collections create [flags]
```

## Exemples

```bash
ktm collections create --operator my-op --parent <parent_uid> --name "Auth regression"
ktm collections create --operator my-op --parent <tests_root> --name "Smoke" -o json | jq .uid
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--name` | string | — | Nom de la nouvelle collection (requis) |
| `--operator` | string | — | Slug de l'opérateur (requis) |
| `--parent` | string | — | UID de la collection parent (requis) |

## Détails

Crée une collection en tant qu'enfant de --parent. L'opérateur et l'UID parent
sont requis ; la nouvelle collection hérite de l'org_id de l'opérateur.
