Renommer une collection

## Utilisation

```bash
ktm collections rename <uid> [flags]
```

## Exemples

```bash
ktm collections rename --operator my-op <uid> --name "Auth regression (v2)"
```

## Arguments

| Argument | Requis |
| --- | --- |
| `uid` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--name` | string | — | Nouveau nom (requis) |
| `--operator` | string | — | Slug de l'opérateur (requis) |

## Détails

Change le nom de la collection. Le parent et le contenu ne sont pas modifiés.
