Lister les tests reusable pour un produit

## Utilisation

```bash
ktm reusables list [flags]
```

## Exemples

```bash
ktm reusables list --product acme~web
ktm reusables list --product acme~web -o table
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--page` | int | — | Numéro de page |
| `--page-size` | int | — | Éléments par page (défaut : 25) |
| `--product` | string | — | Slug du produit (requis) |

## Détails

Résout le reusable_tests_root_collection_uid du produit et liste les éléments
dans cette collection via le service collections.
