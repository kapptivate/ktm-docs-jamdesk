Activer ou désactiver le mode maintenance d'un agent

## Utilisation

```bash
ktm agents maintenance [flags]
```

## Exemples

```bash
ktm agents maintenance --uuid abc-123 --enable --confirm
ktm agents maintenance --uuid abc-123 --disable --confirm
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--confirm` | boolean | — | Exécuter le basculement (sans ce flag, affiche un aperçu) |
| `--disable` | boolean | — | Désactiver le mode maintenance |
| `--enable` | boolean | — | Activer le mode maintenance |
| `--uuid` | string | — | UUID de l'agent (requis) |
