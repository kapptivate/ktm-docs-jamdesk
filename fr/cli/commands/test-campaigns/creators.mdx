Lister les créateurs distincts de Tests campaigns (pour filtrer la liste)

## Utilisation

```bash
ktm test-campaigns creators [flags]
```

## Exemples

```bash
ktm test-campaigns creators --product acme~web
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--operator` | string | — | Slug ou short_name de l'operator |
| `--product` | string | — | Slug ou short_name du produit |

## Détails

Retourne l'ensemble unique des utilisateurs ayant créé des Tests campaigns dans
le périmètre donné (produit ou operator). Utilisez-le pour alimenter le filtre par créateur.
