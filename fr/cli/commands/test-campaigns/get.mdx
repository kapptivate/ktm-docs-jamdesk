Récupérer une Tests campaign avec sa test_list complète

## Utilisation

```bash
ktm test-campaigns get <id> [flags]
```

## Exemples

```bash
ktm test-campaigns get 42
ktm test-campaigns get 42 -o json | jq '.test_list[].name'
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Retourne la forme complète de la Tests campaign, incluant la test_list ordonnée. Chaque
entrée est \{position, test (id), name, owners, input_variables\} : owners et
input_variables sont des surcharges par campaign appliquées uniquement lors de l'exécution.

Pour l'exécuter : ktm ci build --test-campaign &lt;id&gt; | ktm ci run --stdin.
