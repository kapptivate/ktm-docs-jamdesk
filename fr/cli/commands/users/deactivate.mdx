Désactiver un compte utilisateur

## Utilisation

```bash
ktm users deactivate [flags]
```

## Exemples

```bash
# Désactiver l'utilisateur avec l'identifiant 42
ktm users deactivate --id 42
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--id` | int | — | Identifiant de l'utilisateur à désactiver (depuis 'users list') |

## Détails

Désactiver un compte utilisateur par identifiant. Nécessite la permission superadmin ou EDIT_OPERATOR_ADMIN.

Utilisez 'users list' pour trouver les identifiants d'utilisateurs.
