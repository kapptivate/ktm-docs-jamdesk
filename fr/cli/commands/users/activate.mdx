Activer un compte utilisateur

## Utilisation

```bash
ktm users activate [flags]
```

## Exemples

```bash
# Activer l'utilisateur avec l'identifiant 42
ktm users activate --id 42
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--id` | int | — | Identifiant de l'utilisateur à activer (depuis 'users list') |

## Détails

Activer un compte utilisateur par identifiant. Nécessite la permission superadmin ou EDIT_OPERATOR_ADMIN.

Utilisez 'users list' pour trouver les identifiants d'utilisateurs.
