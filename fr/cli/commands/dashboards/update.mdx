Mettre à jour les métadonnées d'un dashboard (nom, bucket, statut favori)

## Utilisation

```bash
ktm dashboards update <uid> [flags]
```

## Exemples

```bash
# Rename a dashboard
ktm dashboards update abc123 --name "New Name"

# Set as favorite
ktm dashboards update abc123 --favorite true
```

## Arguments

| Argument | Requis |
| --- | --- |
| `uid` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--bucket` | string | — | Nouveau bucket InfluxDB |
| `--favorite` | string | — | Définir le statut favori (true/false) |
| `--name` | string | — | Nouveau nom du dashboard |
