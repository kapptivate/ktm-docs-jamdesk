Validation à sec d'un JSON de widget de dashboard

## Utilisation

```bash
ktm dashboards validate-widget [flags]
```

## Exemples

```bash
ktm dashboards validate-widget --json widget.json
cat widget.json | ktm dashboards validate-widget --json -
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--json` | string | — | Chemin du fichier JSON du widget, ou "-" pour stdin (requis) |

## Détails

Valide un fichier JSON de widget sans modifier aucun dashboard.
Vérifie le type, la structure, les champs de requête, la détection de placeholders et les contraintes par type.
