## Détails

Gérer les variables, les groupes de variables et les variants depuis un seul arbre de commandes.

Le système de variables est organisé comme Produit > Variables > Groupes > Variants :
  - Variables  : valeurs typées utilisées par les tests et monitors à l'exécution (single, secret, choice).
  - Groupes    : configurations d'environnement qui contiennent un ensemble de variants.
  - Variants   : déclinaisons d'environnement au sein d'un groupe (ex. dev, staging, prod).

Chaque opération est une sous-commande dédiée à un seul objectif afin que le verbe soit toujours
sans ambiguïté.

<Note>
  Les flags globaux (`--output`, `--debug`, `--host`, ...) s'appliquent à toutes les commandes. Consultez la [vue d'ensemble de la référence des commandes](/fr/cli/commands/overview).
</Note>
