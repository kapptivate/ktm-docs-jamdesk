Désarchiver un test (définit archived=false via PUT)

## Utilisation

```bash
ktm tests unarchive <id> [flags]
```

## Exemples

```bash
ktm tests unarchive 99
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Bascule le flag archived du test à false.

Utilise PATCH avec un corps minimal (uniquement \{archived\}). Cela évite l'aller-retour fragile du payload complet qui peut provoquer une erreur 500 sur les tests dont les types d'action ne sont pas dans le registre du CLI.

## Notes

- Rend visible un test archivé, c'est l'opération inverse de [`ktm tests archive`](/fr/cli/commands/tests/archive).
