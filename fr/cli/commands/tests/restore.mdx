Restaurer un test à une version précédente

## Utilisation

```bash
ktm tests restore <history-id> [flags]
```

## Exemples

```bash
# Trouver les versions de l'historique
ktm tests history 16480 --date 2026-03-06

# Restaurer à une version spécifique
ktm tests restore 42
```

## Arguments

| Argument | Requis |
| --- | --- |
| `history-id` | Oui |

## Détails

Restaurer un test à une version historique par son identifiant d'historique.
Utilisez "tests history &lt;test-id&gt; --date &lt;date&gt;" pour trouver les identifiants d'historique disponibles.

Cette opération est destructive : elle écrase la définition actuelle du test.
