Restaurer un ou plusieurs tests depuis la corbeille

## Utilisation

```bash
ktm tests undelete <id>... [flags]
```

## Exemples

```bash
ktm tests undelete 99
ktm tests undelete 99 100 101
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Récupère les tests depuis la collection test-trash du produit. C'est
l'opération inverse de 'ktm tests delete'. Les tests sont restaurés dans leur
collection précédente.

## Notes

- Restaure les tests depuis la corbeille, c'est l'opération inverse de [`ktm tests delete`](/fr/cli/commands/tests/delete).
