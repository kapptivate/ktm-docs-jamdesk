Archiver un test (définit archived=true via PUT)

## Utilisation

```bash
ktm tests archive <id> [flags]
```

## Exemples

```bash
ktm tests archive 99
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Bascule le flag archived du test à true. Aucun autre champ n'est modifié. Le test reste dans sa collection actuelle. Utilisez 'tests delete' pour le déplacer vers la corbeille à la place.

Utilise PATCH avec un corps minimal (uniquement \{archived\}). Cela évite l'aller-retour fragile du payload complet qui peut provoquer une erreur 500 sur les tests dont les types d'action ne sont pas dans le registre du CLI.

## Notes

- Masque un test sans le supprimer ; réversible avec [`ktm tests unarchive`](/fr/cli/commands/tests/unarchive). Pour déplacer un test vers la corbeille à la place, utilisez [`ktm tests delete`](/fr/cli/commands/tests/delete).
