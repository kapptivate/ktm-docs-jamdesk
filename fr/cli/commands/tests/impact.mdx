## Utilisation

```bash

```

## Exemples


| Argument | Requis |

## Détails

Vérifier l'impact en aval d'un test : quelles tests campaigns l'incluent et quels monitors
l'utilisent. Utile avant de supprimer, archiver ou modifier significativement un test.
