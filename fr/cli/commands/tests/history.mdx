Lister l'historique des versions d'un test

## Utilisation

```bash
ktm tests history <id> [flags]
```

## Exemples

```bash
# Lister l'historique des versions
ktm tests history 16480

# Filtrer par date
ktm tests history 16480 --date 2026-03-06

# Lister l'historique en tableau
ktm tests history 16480 -o table
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--date` | string | — | Filtrer les entrées d'historique par date (YYYY-MM-DD) |

## Détails

Lister les entrées de l'historique des versions pour un test. Chaque entrée contient un identifiant d'historique
qui peut être passé à 'ktm tests restore' pour revenir à cette version.
