Lister les tests, avec filtrage optionnel par operator et produit

## Utilisation

```bash
ktm tests list [flags]
```

## Exemples

```bash
# Lister les tests pour un produit
ktm tests list --product acme~web

# Lister les tests archivés
ktm tests list --product acme~web --archived true

# Lister les tests avec un nom contenant "login"
ktm tests list --product acme~web --name login

# Format tableau
ktm tests list --product acme~web -o table
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--archived` | string | — | Filtrer par statut archivé : true ou false |
| `--name` | string | — | Filtrer par nom de test (correspondance partielle) |
| `--operator` | string | — | Slug de l'operator pour limiter les résultats |
| `--page` | int | — | Numéro de page |
| `--page-size` | int | — | Éléments par page |
| `--product` | string | — | Slug du produit pour limiter les résultats |
