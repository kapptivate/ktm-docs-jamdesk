Forcer l'exécution immédiate d'un monitor, en dehors de son planning normal

## Utilisation

```bash
ktm monitors force-run <id> [flags]
```

## Exemples

```bash
ktm monitors force-run 42
ktm monitors force-run 42 -o json
```

## Arguments

| Argument | Requis |
| --- | --- |
| `id` | Oui |

## Détails

Déclenche une exécution immédiate hors planning d'un monitor.

Le monitor s'exécute une fois immédiatement, indépendamment de son planning configuré.
Utilisez 'monitoring-results list' avec --monitor pour suivre le statut de l'exécution.
