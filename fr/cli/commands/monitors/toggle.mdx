Activer ou désactiver des monitors en masse

## Utilisation

```bash
ktm monitors toggle [flags]
```

## Exemples

```bash
ktm monitors toggle --ids 42,43 --active false
ktm monitors toggle --ids 42 --active true
```

## Flags

| Flag | Type | Défaut | Description |
| --- | --- | --- | --- |
| `--active` | string | — | true pour activer, false pour désactiver (requis) |
| `--ids` | string | — | Identifiants de monitors séparés par des virgules à basculer (requis) |
