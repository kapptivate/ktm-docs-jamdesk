## Détails

Gérer les agents (le matériel/logiciel qui exécute les ressources). Portée opérateur.

Trois types d'agents :
  cellular    Passerelles GSM avec des slots de cartes SIM
  ethernet    Agents web autonomes (pas de slots, capacité = max de tests parallèles)
  smartphone  Hubs téléphoniques avec smartphones connectés

Utilisez 'resources list' pour obtenir les informations d'appareils au niveau produit.
