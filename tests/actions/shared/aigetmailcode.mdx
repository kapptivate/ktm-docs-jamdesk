---
title: AI get mail code
description: Extract an OTP or verification code from email using AI.
---

Use AI to extract OTP (one-time password) or verification codes from emails received at a specified mailbox. This integrates with Kapptivate's mail service to retrieve the most recent emails and identify numeric codes for two-factor authentication flows.

## Channels

- ✅ Web
- ✅ Smartphone (Android/iOS)
- ✅ Cellular

All channels support mail code extraction equally.

## Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| mailbox | Template | Yes | Email address or mailbox identifier to check for codes. Supports templating (e.g., `${TEST_EMAIL}`). |
| variable | String | No | Variable name to store extracted code. If omitted, code is only logged. |
| mocked_response | Object | No | Mock response for testing (JSON). Use in development/testing only. |

## Example

```json
{
  "type": "aigetmailcode",
  "version": "v1.0.0",
  "inputs": {
    "mailbox": "test@example.com",
    "variable": "otp_code"
  }
}
```

Using a test variable for mailbox:

```json
{
  "type": "aigetmailcode",
  "inputs": {
    "mailbox": "${VERIFICATION_EMAIL}",
    "variable": "two_factor_code"
  }
}
```

## Notes

- Retrieves recent emails from the mailbox.
- Uses AI to identify and extract numeric verification codes.
- Extracted code is stored in the specified variable for use in subsequent steps.
- Requires mail service integration to be configured.
- Common uses: complete 2FA flows, test OTP authentication, verify email-based verification.
