---
title: Select
description: Legacy step that picks an option in a native select dropdown.
---

<Warning>
  This step no longer appears in the step palette. Existing tests that use it keep running. For new tests, use [Click](/tests/actions/web/click) to open the dropdown and click the option, which also works for custom (non-native) dropdowns.
</Warning>

Pick an option in a native `<select>` element, by visible text, by position, or by attribute.

## Parameters

| Parameter | JSON field | Required | Description |
|-----------|------------|----------|-------------|
| Target element | `locator.description` | Yes, unless a manual selector is set | Natural-language description of the select element. |
| Manual selector | `locator.selector` | No | XPath or CSS selector, used when AI is disabled or as the saved selector. |
| Select option by | `by` | Yes | `visible_text` (default), `index`, or `attribute`. |
| Value | `visible_text.value` | When by is `visible_text` | The option's visible text. |
| Index | `index.value` | When by is `index` | The option's position, starting at 0. |
| Attribute name and value | `attribute.name`, `attribute.value` | When by is `attribute` | The HTML attribute to match, for example `value` = `FR`. |
| AI usage | `ai_strategy` | No | `inherit` (default, follows the test strategy), `smart`, or `always`. |
| Ignore error on this step | `ignore_errors` | No | The run continues with the next step if this one fails. Default: off. |

## In the scenario JSON

The step lives in `step_groups[].steps` of the test scenario. Only the sub-object matching `by` is kept.

```json
{
  "id": "step-1",
  "type": "select",
  "version": "v1.0.0",
  "disabled": false,
  "inputs": {
    "locator": {
      "selector": [{ "type": "value", "data": { "value": "" } }],
      "description": [{ "type": "value", "data": { "value": "The country dropdown in the shipping form" } }]
    },
    "by": "visible_text",
    "visible_text": {
      "value": [{ "type": "value", "data": { "value": "France" } }]
    },
    "enable_ai": true,
    "ai_strategy": "inherit",
    "ignore_errors": false
  },
  "use_capabilities": false,
  "capabilities": {}
}
```

## Results and metrics

- **Timing**: start, end, and duration (in seconds) recorded in the step result.
- **Status**: failed if the element or the requested option cannot be found.
- **Screenshot**: captured after the selection.
