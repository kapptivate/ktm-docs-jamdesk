---
title: Web recorder
description: Record a web journey in Chrome and turn it into a test script.
---

The Kapptivate web recorder is a Chrome extension that records a browsing session, clicks, typing, navigation, and exports it as a script you can paste into a test. Walk through your scenario once in the browser, and the test writes itself.

## Install the extension

The recorder ships as an unpacked Chrome extension (`kapptivate-web-recorder.zip`):

<Steps>
  <Step title="Open Chrome's Extensions page">
    Download and unzip the package, then navigate to `chrome://extensions`.

    <Frame caption="The Chrome Extensions page">
      ![Chrome Extensions page](/images/recorders/web-01.webp)
    </Frame>
  </Step>
  <Step title="Load the extension">
    Enable **Developer mode** (top right), click **Load unpacked**, and select the unzipped folder. The Kapptivate icon appears in the toolbar.

    <Frame caption="The Kapptivate icon in the Chrome toolbar">
      ![Kapptivate icon in the toolbar](/images/recorders/web-02.webp)
    </Frame>
  </Step>
</Steps>

## Record a test

Click the Kapptivate icon to open the recorder:

<Frame caption="The web recorder main view">
  ![Web recorder main view](/images/recorders/web-03.webp)
</Frame>

1. Click **Record**, navigate to your starting page, and walk through the scenario: every click and input is captured as an action.
2. Click the icon again to review the recorded actions.
3. Click **Stop**, then **Export** and **CopyToClipboard**.

<Frame caption="Stop the recording, then export it to the clipboard">
  ![Stop and export buttons](/images/recorders/web-04.webp)
</Frame>

<Frame caption="The export panel">
  ![Export and CopyToClipboard](/images/recorders/web-05.webp)
</Frame>

## Import the recording

In your workspace, create a test with a **Web Script** action:

<Frame caption="Importing the recording into a Web Script action">
  ![Web Script action](/images/recorders/web-06.webp)
</Frame>

Select the default Ethernet agent, switch to **raw script**, and paste the recording:

<Frame caption="Paste the script in raw script mode">
  ![Raw script mode](/images/recorders/web-07.webp)
</Frame>

Save and run the test once before wiring it into a monitor.

<Note>
  The recorder generates a [script action](/tests/actions/legacy/script-actions): JavaScript you can then edit freely, add assertions to, or enrich with [variables and custom metrics](/tests/actions/legacy/script-actions-variables).
</Note>
